# sew 🍳  
> 一个基于 AI 生成的静态菜谱应用原型（uni-app）

- 在线地址：https://ang-l.github.io/sew

## 📌 项目简介

**sew** 是一个以「做饭体验」为核心的静态菜谱项目，整体流程由 AI 辅助完成：

- 原型设计与 UI：由 Google Stitch 生成  
- 页面代码生成：由 Google AI Studio 生成  
- 技术框架：uni-app  
- 项目定位：可直接使用 / 可二次开发 / 学习参考  

该项目不依赖后端服务，开箱即用，适合：

- uni-app 学习者  
- 菜谱 / 内容类产品原型  
- AI 生成项目实践参考  
- 快速二次开发接入 API  

---

## 📦 数据来源

菜谱基础数据采集自：

https://github.com/Gar-b-age/CookLikeHOC  

采集内容包括：

- 菜品名称  
- 菜品图片（图片名称）  
- 食材列表  
- 制作步骤 / 流程  

---

## 🧠 AI 生成字段说明

以下字段由 AI 根据食材、步骤等信息生成，仅供展示与参考，不代表真实或权威数据：

| 字段 | 说明 |
|------|------|
| `tagline` | 菜品一句话描述 |
| `prepTime` | 预计制作时间 |
| `difficulty` | 制作难度（简单 / 中等 / 困难） |
| `calories` | 预估热量（大致范围） |

⚠️ 以上数据不具备营养或专业参考价值。
